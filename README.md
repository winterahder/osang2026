## 🔑 학번 체계

`[학년(1)][반(2)][번호(2)]` = 5자리  
예) `30101` → 3학년 1반 1번

### 3개년 학번 매핑 (students.json 내 필드)

| 학생 | hak | g1_hak | g2_hak | g3_hak |
|---|---|---|---|---|
| 현 3학년 권명승 | 30101 | 10801 | 20101 | 30101 |
| 현 2학년 김강현 | 20102 | 10604 | 20102 | — |

> 반/번호가 매년 바뀌어도 **학년별 학번으로 과거 성적 매칭이 가능**합니다.

## 🛠️ 데이터 추가 방법

각 탭의 데이터는 `data/{분류}.json` 파일에 아래 형식으로 추가하면 자동 반영됩니다:

```json
{
  "30101": [
    { "학년": 1, "학기": 1, "과목": "국어", "등급": 2, "원점수": 88 }
  ],
  "30102": [ ... ]
}
```

키는 학생의 **현재 학번(hak)**입니다.
