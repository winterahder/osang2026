# 2026 대입 상담카드

## 🚀 배포
https://winterahder.github.io/osang2026/

## 📁 구조
```
osang2026/
├── index.html
├── css/styles.css
├── js/app.js
└── data/
    ├── meta.json         # 메타정보
    ├── students.json     # 학생 명단 (하드코딩)
    ├── mock.json         # 모의고사 성적
    ├── grades.json       # 내신 (추후)
    ├── sewtuk.json       # 세특 (추후)
    ├── behavior.json     # 행동특성 (추후)
    ├── autonomy.json     # 자율활동
    ├── career.json       # 진로활동
    ├── club.json         # 동아리
    ├── volunteer.json    # 봉사
    ├── award.json        # 수상 (참고용)
    ├── reading.json      # 독서 (참고용)
    ├── applications.json # 지원대학
    ├── checklist.json    # 체크리스트
    └── attendance.json   # 출결
```

## ✨ 주요 기능
- 학년/반 2단계 필터 + 이름·학번 검색
- 모의고사 등급/백분위/표준점수/원점수 전환 표시
- 회차별 차트 + 세부표
- 1·2·3학년 학번 이력 추적

## 🔐 개인정보
- 주소, 주민번호, 연락처 미포함
- 공개 저장소 주의
